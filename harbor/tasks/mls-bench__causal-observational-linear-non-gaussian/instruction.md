# MLS-Bench: causal-observational-linear-non-gaussian

# Causal Discovery: Observational Linear Non-Gaussian Data (LiNGAM)

## Research Question
Implement a causal discovery algorithm that recovers the **directed DAG**
from purely observational data generated by a Linear Non-Gaussian Acyclic Model
(LiNGAM).

## Background
LiNGAM-based methods exploit non-Gaussian noise to achieve full DAG
identifiability from observational data alone, going beyond the Markov
Equivalence Class limit of constraint-based (PC) and score-based (GES) methods.
The original identifiability result is due to Shimizu, Hoyer, Hyvarinen &
Kerminen, "A Linear Non-Gaussian Acyclic Model for Causal Discovery,"
JMLR 7, 2006.

The benchmark uses synthetic Erdos-Renyi and scale-free DAGs with signed linear
edge weights and independent non-Gaussian noise. The method should handle
moderate-to-large node counts and avoid assuming a single fixed noise
distribution or graph family.

## Task
Implement `run_causal_discovery(X)` in `bench/custom_algorithm.py`. It must
return a directed DAG (skeleton + correct edge orientation) compatible with the
benchmark evaluation.

## Evaluation Scenarios

| Label  | Graph type      | Nodes | Samples | Noise       |
|--------|-----------------|-------|---------|-------------|
| ER30   | Erdos-Renyi     | 30    | 1000    | Laplace     |
| ER50   | Erdos-Renyi     | 50    | 2000    | Exponential |
| SF100  | Scale-Free (BA) | 100   | 1000    | Uniform     |

## Metrics
Computed on the directed edge set (skeleton + direction must both be correct):
- **F1** (primary ranking metric, higher is better)
- **SHD** (lower is better)
- **Precision** (higher is better)
- **Recall** (higher is better)

## Reference baselines
- `icalingam`: ICA-based LiNGAM. Shimizu, Hoyer, Hyvarinen & Kerminen,
  "A Linear Non-Gaussian Acyclic Model for Causal Discovery," JMLR 7, 2006.
- `directlingam`: DirectLiNGAM. Shimizu et al., "DirectLiNGAM: A Direct Method
  for Learning a Linear Non-Gaussian Structural Equation Model," JMLR 12, 2011
  (arXiv:1101.2489).
- `notears`: NOTEARS. Zheng, Aragam, Ravikumar & Xing, "DAGs with NO TEARS:
  Continuous Optimization for Structure Learning," NeurIPS 2018
  (arXiv:1803.01422). Continuous score-based DAG learning via a smooth
  acyclicity constraint.

The scientific contribution should be a modular method for exploiting
non-Gaussian observational structure, such as independent component ideas,
direct causal ordering, continuous optimization, hybrid screening, or a new
directed-graph recovery procedure.

    41:             if len(row_index_list) == 0:
    42:                 break

    44:             causal_order.append(original_index[target_index])
    45:             original_index = np.delete(original_index, target_index, axis=0)
    46:             mask = np.delete(np.arange(len(matrix)), target_index, axis=0)
    47:             matrix = matrix[mask][:, mask]

    49:             return None

    60:         for i, j in pos_list[initial_zero_num:]:

    64:                 break

    11:     """
    12:     import os
    13:     import numpy as np
    14:     from sklearn.utils import check_array
    15:     from causallearn.search.FCMBased.lingam.base import _BaseLiNGAM
    16:

    19:     n_features = X.shape[1]
    20: 
    21:     # Core DirectLiNGAM steps (from causallearn.search.FCMBased.lingam.direct_lingam)
    22:     def _residual(xi: np.ndarray, xj: np.ndarray) -> np.ndarray:
    23:         return xi - (np.cov(xi, xj)[0, 1] / np.var(xj)) * xj
    24: 
    25:     def _entropy(u: np.ndarray) -> float:
    26:         k1 = 79.047
    27:         k2 = 7.4129
    28:         gamma = 0.37457
    29:         return (1 + np.log(2 * np.pi)) / 2 -                k1 * (np.mean(np.log(np.cosh(u))) - gamma) ** 2 -                k2 * (np.mean(u * np.exp((-u ** 2) / 2))) ** 2
    30: 
    31:     def _diff_mutual_info(
    32:         xi_std: np.ndarray,
    33:         xj_std: np.ndarray,
    34:         ri_j: np.ndarray,
    35:         rj_i: np.ndarray,
    36:     ) -> float:
    37:         return (_entropy(xj_std) + _entropy(ri_j / np.std(ri_j))) -                (_entropy(xi_std) + _entropy(rj_i / np.std(rj_i)))
    38: 
    39:     def _search_causal_order(X_work: np.ndarray, U: np.ndarray) -> int:
    40:         if len(U) == 1:
    41:             return int(U[0])
    42:         M_list = []
    43:         for i in U:
    44:             M = 0.0
    45:             for j in U:
    46:                 if i == j:
    47:                     continue
    48:                 xi_std = (X_work[:, i] - np.mean(X_work[:, i])) / np.std(X_work[:, i])
    49:                 xj_std = (X_work[:, j] - np.mean(X_work[:, j])) / np.std(X_work[:, j])
    50:                 ri_j = _residual(xi_std, xj_std)
    51:                 rj_i = _residual(xj_std, xi_std)
    52:                 M += np.min([0.0, _diff_mutual_info(xi_std, xj_std, ri_j, rj_i)]) ** 2
    53:             M_list.append(-1.0 * M)

### `notears` baseline — editable region  [READ-ONLY — reference implementation]

In `causal-learn/bench/custom_algorithm.py`:

```python
Lines 3–83:
     1: import numpy as np
     2: 
     3: # =====================================================================
     4: # EDITABLE: implement run_causal_discovery below
     5: # =====================================================================
     6: def run_causal_discovery(X: np.ndarray) -> np.ndarray:
     7:     """
     8:     Input:  X of shape (n_samples, n_variables)
     9:     Output: adjacency matrix B of shape (n_variables, n_variables)
    10:             B[i, j] != 0  means j -> i  (follows causal-learn convention)
    11:     """
    12:     import numpy as np
    13:     import scipy.linalg as sla
    14:     from scipy.optimize import minimize
    15:     from sklearn.utils import check_array
    16: 
    17:     X = check_array(X)
    18:     n, d = X.shape
    19: 
    20:     # Reference defaults from Zheng et al. 2018 reference impl.
    21:     lambda1 = 0.1
    22:     max_iter = 100
    23:     h_tol = 1e-8
    24:     rho_max = 1e16
    25:     w_threshold = 0.3
    26: 
    27:     def _loss_and_grad(W):
    28:         # Squared-error regression loss: 1/(2n) * ||X - X W||^2
    29:         R = X - X @ W
    30:         loss = 0.5 / n * (R ** 2).sum()
    31:         G = -1.0 / n * X.T @ R
    32:         return loss, G
    33: 
    34:     def _h_and_grad(W):
    35:         # h(W) = tr(exp(W*W)) - d  (Zheng 2018 smooth acyclicity)
    36:         M = W * W
    37:         E = sla.expm(M)
    38:         h = np.trace(E) - d
    39:         G = E.T * 2 * W
    40:         return h, G
    41: 
    42:     def _obj(w_pm, rho, alpha):
    43:         w_pm = w_pm.reshape(2, d * d)
    44:         W = (w_pm[0] - w_pm[1]).reshape(d, d)
    45:         loss, G_loss = _loss_and_grad(W)
    46:         h, G_h = _h_and_grad(W)
    47:         obj = loss + 0.5 * rho * h * h + alpha * h + lambda1 * w_pm.sum()
    48:         G_smooth = G_loss + (rho * h + alpha) * G_h
    49:         g = np.concatenate([
    50:             (G_smooth + lambda1).flatten(),
    51:             (-G_smooth + lambda1).flatten(),
    52:         ])
    53:         return obj, g
    54: 
    55:     w_est = np.zeros(2 * d * d)
    56:     rho, alpha, h = 1.0, 0.0, np.inf
    57:     # Non-negative bounds; force diagonal to zero (no self-loops)
    58:     bnds = [(0, 0) if (i == j) else (0, None)
    59:             for _ in range(2) for i in range(d) for j in range(d)]
    60: 
    61:     for _ in range(max_iter):
    62:         while rho < rho_max:
    63:             sol = minimize(_obj, w_est, args=(rho, alpha),
    64:                            method='L-BFGS-B', jac=True, bounds=bnds)
    65:             w_new = sol.x
    66:             W_new = (w_new[:d * d] - w_new[d * d:]).reshape(d, d)
    67:             h_new, _ = _h_and_grad(W_new)
    68:             if h_new > 0.25 * h:
    69:                 rho *= 10
    70:             else:
    71:                 break
    72:         w_est, h = w_new, h_new
    73:         alpha += rho * h
    74:         if h <= h_tol or rho >= rho_max:
    75:             break
    76: 
    77:     W_final = (w_est[:d * d] - w_est[d * d:]).reshape(d, d)
    78:     W_final[np.abs(W_final) < w_threshold] = 0.0
    79: 
    80:     # NOTEARS: W[i, j] != 0 means i -> j
    81:     # causal-learn / this task: B[i, j] != 0 means j -> i. Transpose.
    82:     return W_final.T
    83: # =====================================================================
```
