# Causal Discovery: Observational Nonlinear Data (Additive Noise Models)

## Research Question
Implement a causal discovery algorithm that recovers the **directed DAG** from
purely observational data generated by a nonlinear Additive Noise Model (ANM).

## Background
Additive Noise Models assume structural equations of the form
`x_j = f_j(parents(j)) + e_j`, where `f_j` are nonlinear functions and `e_j`
are independent noise terms. Under mild conditions on `f_j` and the noise
distribution, the causal DAG is identifiable from observational data alone
(Hoyer, Janzing, Mooij, Peters & Scholkopf, "Nonlinear causal discovery with
additive noise models," NeurIPS 2008; Peters, Mooij, Janzing & Scholkopf,
"Causal Discovery with Continuous Additive Noise Models," JMLR 15, 2014). This
goes beyond linear methods (LiNGAM, PC, GES) that assume linearity.

Key challenges include: (1) testing independence in high-dimensional nonlinear
settings, (2) scaling to larger graphs, (3) handling diverse nonlinear function
types (MLP, GP, polynomial, sigmoid), and (4) robustness when noise is
Gaussian, where ANM identifiability becomes more delicate.

## Task
Implement `run_causal_discovery(X)` in `bench/custom_algorithm.py`. It must
return a directed DAG compatible with the benchmark evaluation.

- **SHD** (lower is better)
- **Precision**, **Recall** (higher is better)

## Reference baselines
- `cam`: Causal Additive Models with GAM regression. Score-based. Buhlmann,
  Peters & Ernest, "CAM: Causal additive models, high-dimensional order search
  and penalized regression," Annals of Statistics 42(6), 2014
  (arXiv:1310.1533).
- `notears_mlp`: NOTEARS with MLP parametrization for nonlinear additive
  structure. Continuous optimization. Zheng, Dan, Aragam, Ravikumar & Xing,
  "Learning Sparse Nonparametric DAGs," AISTATS 2020 (arXiv:1909.13189). The
  default uses a one-hidden-layer MLP with sigmoid activation as in the paper.
- `directlingam`: Linear reference baseline. Shimizu et al., JMLR 12, 2011
  (arXiv:1101.2489).
- `grandag`: GraN-DAG. Lachapelle, Brouillard, Deleu & Lacoste-Julien,
  "Gradient-Based Neural DAG Learning," ICLR 2020 (arXiv:1906.02226).
  Gradient-based DAG learning using neural conditional densities with an
  augmented-Lagrangian acyclicity constraint.

Suitable contributions include nonlinear independence tests, additive-noise
scoring, differentiable DAG learning, hybrid ordering methods, or other
modular procedures that exploit nonlinear causal asymmetry while remaining
practical at the evaluated graph sizes.

    23:     w_threshold = 0.3

    25: 
    26:     # Standardize data
    27:     X_std = (X - X.mean(axis=0)) / (X.std(axis=0) + 1e-8)
    28: 
    29:     # --- NOTEARS (linear) on the data ---
    30:     # Formulation: minimize 0.5/n * ||X - X W||^2_F + lambda1 * |W|_1
    31:     # subject to h(W) = tr(e^{W o W}) - d = 0
    32: 
    33:     def _h(W):
    34:         """Acyclicity constraint: h(W) = tr(e^{W o W}) - d."""
    35:         M = W * W
    36:         # Matrix exponential trace via power series (consistent with _h_grad)
    37:         expm_M = np.eye(n_vars)
    38:         power = np.eye(n_vars)
    39:         for k in range(1, 12):
    40:             power = power @ M / k
    41:             expm_M += power
    42:         return np.trace(expm_M) - n_vars
    43: 
    44:     def _h_grad(W):
    45:         """Gradient of h w.r.t. W."""
    46:         M = W * W
    47:         # expm(M) via series (10 terms)
    48:         expm_M = np.eye(n_vars)
    49:         power = np.eye(n_vars)
    50:         for k in range(1, 12):
    51:             power = power @ M / k
    52:             expm_M += power
    53:         return 2 * W * expm_M
    54: 
    55:     def _loss_and_grad(W_flat, rho, alpha):
    56:         W = W_flat.reshape(n_vars, n_vars)
    57:         # Zero diagonal (no self-loops)
    58:         np.fill_diagonal(W, 0)
    59: 
    60:         # MSE loss: 0.5/n * ||X - XW||^2
    61:         R = X_std - X_std @ W  # (n, d)
    62:         loss = 0.5 / n_samples * np.sum(R ** 2)
    63:         # Gradient of MSE w.r.t. W
    64:         G_mse = -1.0 / n_samples * (X_std.T @ R)  # (d, d)
    65: 
    66:         # L1 penalty
    67:         l1_loss = lambda1 * np.sum(np.abs(W))
    68:         G_l1 = lambda1 * np.sign(W)
    69: 
    70:         # Acyclicity
    71:         h_val = _h(W)
    72:         G_h = _h_grad(W)
    73: 
    74:         total_loss = loss + l1_loss + 0.5 * rho * h_val ** 2 + alpha * h_val
    75:         G_total = G_mse + G_l1 + (rho * h_val + alpha) * G_h

    77:         # Zero diagonal gradient
    78:         np.fill_diagonal(G_total, 0)

    96:             jac=True,
    97:             options={'maxiter': 500}

   102:         h_new = _h(W_est)

   104:             rho *= 10.0
   105:         alpha_dual += rho * h_new
   106:         h_prev = h_new
   107: 
   108:         if abs(h_new) < h_tol or rho > rho_max:

   112:     # Use the linear NOTEARS skeleton and refine with nonlinear regression
   113:     from sklearn.ensemble import GradientBoostingRegressor

   115:     # Threshold the linear result
   116:     W_abs = np.abs(W_est)

     7:     """GraN-DAG (Lachapelle et al., ICLR 2020).
     8: 
     9:     B[i,j] != 0 means j -> i (causal-learn convention).
    10:     """
    11:     import os
    12:     import torch, torch.nn as nn, torch.nn.functional as F
    13:     from torch import distributions
    14: 
    15:     seed = int(os.environ.get("SEED", "42"))

    17:     torch.manual_seed(seed)
    18:     np.random.seed(seed)
    19: 
    20:     n, d = X.shape
    21:     DT = torch.float64
    22: 
    23:     # ================================================================== #
    24:     # Per-variable MLP model (NonlinearGaussANM, 2x10, leaky-relu)       #
    25:     # ================================================================== #
    26:     class _M(nn.Module):
    27:         def __init__(self):
    28:             super().__init__()
    29:             # Explicit adjacency mask (not a Parameter -- updated by clamping only)
    30:             self.adjacency = torch.ones(d, d, dtype=DT) - torch.eye(d, dtype=DT)
    31:             layers = [d, 10, 10, 1]  # [input, hidden1, hidden2, output_dim]
    32:             self.wt = nn.ParameterList()
    33:             self.bi = nn.ParameterList()
    34:             for k in range(len(layers) - 1):
    35:                 self.wt.append(nn.Parameter(
    36:                     torch.zeros(d, layers[k + 1], layers[k], dtype=DT)))
    37:                 self.bi.append(nn.Parameter(
    38:                     torch.zeros(d, layers[k + 1], dtype=DT)))
    39:             # Per-variable learnable noise log-std (ANM model)
    40:             self.log_std = nn.ParameterList(
    41:                 [nn.Parameter(torch.zeros(1, dtype=DT)) for _ in range(d)])
    42:             # Xavier init matching gcastle's reset_params() order
    43:             g = nn.init.calculate_gain('leaky_relu')
    44:             with torch.no_grad():
    45:                 for nd in range(d):
    46:                     for w in self.wt:
    47:                         nn.init.xavier_uniform_(w[nd], gain=g)
    48:                     for b in self.bi:
    49:                         b[nd].zero_()
    50: 
    51:         def _fwd(self, x):
    52:             """Per-variable forward pass with adjacency masking on first layer."""
    53:             for k in range(3):
    54:                 if k == 0:
    55:                     x = torch.einsum("tij,ljt,bj->bti", self.wt[k],
    56:                                      self.adjacency.unsqueeze(0), x) \
    57:                         + self.bi[k]
    58:                 else:
    59:                     x = torch.einsum("tij,btj->bti", self.wt[k], x) \
    60:                         + self.bi[k]
    61:                 if k < 2:
    62:                     x = F.leaky_relu(x)
    63:             return torch.unbind(x, 1)  # d tensors of (batch, 1)
    64: 
    65:         def log_lik(self, x, detach_target=False):
    66:             """(batch, d) per-variable Gaussian log-likelihoods."""
    67:             preds = self._fwd(x)
    68:             parts = []
    69:             for i in range(d):
    70:                 mu = preds[i].squeeze(1)
    71:                 sig = torch.exp(self.log_std[i])
    72:                 xi = x[:, i].detach() if detach_target else x[:, i]
    73:                 parts.append(
    74:                     distributions.Normal(mu, sig).log_prob(xi).unsqueeze(1))
    75:             return torch.cat(parts, 1)
    76: 
    77:         def w_adj(self):
    78:             """Weighted adjacency via product of |weights|, path-normalised."""
    79:             prod = torch.eye(d, dtype=DT)
    80:             pn = torch.eye(d, dtype=DT)
    81:             off = (1.0 - torch.eye(d, dtype=DT)).unsqueeze(0)
    82:             for i, w in enumerate(self.wt):
    83:                 wa = torch.abs(w)
    84:                 if i == 0:
    85:                     prod = torch.einsum("tij,ljt,jk->tik",
    86:                                         wa, self.adjacency.unsqueeze(0), prod)
    87:                     pn = torch.einsum("tij,ljt,jk->tik",
    88:                                       torch.ones_like(wa), off, pn)
    89:                 else:
    90:                     prod = torch.einsum("tij,tjk->tik", wa, prod)
    91:                     pn = torch.einsum("tij,tjk->tik",
    92:                                       torch.ones_like(wa), pn)
    93:             prod = prod.sum(1)
    94:             pn = pn.sum(1)
    95:             return (prod / (pn + torch.eye(d, dtype=DT))).t()

   100:     # Data split (80/20, no shuffle, no normalise -- gcastle defaults)    #

   102:     tn = int(n * 0.8)
   103:     Xtr = torch.as_tensor(X[:tn], dtype=DT)

   105:     rng_tr = np.random.RandomState(seed)
   106:     rng_te = np.random.RandomState(seed + 1)
   107: 
   108:     def _samp(data, rng, bs):

   110:         return data[torch.as_tensor(idx).long()]
   111: 
   112:     # ================================================================== #
   113:     # Augmented-Lagrangian training (convergence-based mu/lambda update)  #
   114:     # ================================================================== #
   115:     mu, lamb = 0.001, 0.0        # penalty & dual variable
   116:     opt = torch.optim.RMSprop(mdl.parameters(), lr=0.001)
   117:     a_val, nns, hh = [], [], []  # validation AL, not-nll, constraint history
   118:     BS, ITER, WIN = min(64, tn), 30000, 100
   119: 
   120:     for it in range(ITER):
   121:         mdl.train()
   122:         xb = _samp(Xtr, rng_tr, BS)
   123:         loss = -mdl.log_lik(xb).mean()
   124:         mdl.eval()
   125: 
   126:         wa = mdl.w_adj()
   127:         h = torch.trace(torch.matrix_exp(wa)) - d
   128:         al = loss + 0.5 * mu * h ** 2 + lamb * h
   129: 
   130:         opt.zero_grad()
   131:         al.backward()
   132:         opt.step()
   133: 
   134:         # Edge clamping -- only apply periodically to avoid premature
   135:         # irreversible edge removal that causes instability across runs.
   136:         # gcastle default threshold is 1e-4, but applying every step is
   137:         # too aggressive; applying every 500 iterations with a stricter
   138:         # weight threshold (1e-3) is more stable.
   139:         if it > 0 and it % 500 == 0:
   140:             with torch.no_grad():
   141:                 mdl.adjacency *= (wa > 1e-3).to(DT)
   142: 
   143:         nns.append(0.5 * mu * h.item() ** 2 + lamb * h.item())
   144: 
   145:         # Validation every WIN iterations
   146:         if it % WIN == 0:
   147:             with torch.no_grad():
   148:                 vl = -mdl.log_lik(
   149:                     _samp(Xte, rng_te, Xte.shape[0])).mean()
   150:                 a_val.append([it, vl.item() + nns[-1]])
   151: 
   152:         # Convergence delta (checked every 2*WIN)
   153:         dl = -np.inf
   154:         if it >= 2 * WIN and it % (2 * WIN) == 0:
   155:             t0, th, t1 = a_val[-3][1], a_val[-2][1], a_val[-1][1]
   156:             if not (min(t0, t1) < th < max(t0, t1)):
   157:                 dl = -np.inf
   158:             else:
   159:                 dl = (t1 - t0) / WIN

   162:         if h.item() > 1e-8:

   164:                 lamb += mu * h.item()

   166:                 if len(hh) >= 2 and hh[-1] > hh[-2] * 0.9:
   167:                     mu *= 10
   168:                 # Adjust moving-average validation to account for new mu/lambda
   169:                 gap = (0.5 * mu * h.item() ** 2
   170:                        + lamb * h.item() - nns[-1])
   171:                 a_val[-1][1] += gap
   172:                 opt = torch.optim.RMSprop(mdl.parameters(), lr=0.001)
   173:         else:
   174:             # Converged -- final clamping of zero-weight edges
   175:             with torch.no_grad():
   176:                 mdl.adjacency *= (wa > 0).to(DT)
   177:             break

   179:     # ================================================================== #
   180:     # DAG enforcement: Jacobian threshold + weakest-edge removal          #
   181:     # ================================================================== #
   182:     mdl.eval()

   184:     ll = mdl.log_lik(xj, detach_target=True)       # (tn, d)
   185:     lps = torch.unbind(ll, 1)                       # d tensors of (tn,)
   186:     jac = torch.zeros(d, d, dtype=DT)
   187:     for i in range(d):
   188:         g = torch.autograd.grad(
   189:             lps[i], xj, retain_graph=True,
   190:             grad_outputs=torch.ones(Xtr.shape[0], dtype=DT))[0]
   191:         jac[i] = g.abs().mean(0)
   192:     A = jac.t().detach().numpy()

   194:     # Find smallest threshold that produces an acyclic graph
   195:     with torch.no_grad():
   196:         for thr in np.unique(A):
   197:             keep = torch.tensor(A > thr + 1e-8, dtype=DT)
   198:             na = mdl.adjacency * keep
   199:             # Acyclicity check via matrix-power trace
   200:             prod = torch.eye(d, dtype=DT)
   201:             ok = True
   202:             for _ in range(d):
   203:                 prod = na @ prod
   204:                 if prod.trace() != 0:
   205:                     ok = False
   206:                     break
   207:             if ok:

   209:                 break

## Tips

- Keep the function/class signatures of the editable regions identical;
  evaluation imports them by name.
- Determinism matters: seeds are fixed; don't introduce hidden randomness.
- The baseline implementations above are deliberately strong. Aim for an
  *algorithmic* improvement — many hyperparameters are locked outside the
  editable surface anyway.

Good luck.
