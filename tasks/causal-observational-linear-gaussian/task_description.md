# Causal Discovery from Linear Gaussian Observations (CPDAG Recovery)

## Research Question
Implement a causal discovery algorithm that recovers the **CPDAG** from purely
observational data generated by a linear Gaussian structural equation model
(SEM).

## Background
Under linear Gaussian assumptions, observational data generally identifies
only the Markov Equivalence Class (MEC) of the true DAG, not a unique fully
directed DAG. The MEC is represented by a CPDAG. The benchmark studies
robustness across sparse and denser graphs and across noise levels, and the
method should not rely on dataset-specific constants.

## Metrics
Computed by comparing the estimated CPDAG with the ground-truth CPDAG:
- **SHD** via `SHD(...).get_shd()` (lower is better)
- **Adjacency Precision / Recall** via `AdjacencyConfusion` (higher is better)
- **Arrow Precision / Recall** via `ArrowConfusion` (higher is better)

## Reference baselines
Citations are provided so the agent can study the prior art; default
hyperparameters are the ones recommended in the cited papers.

- `pc`: Peter-Clark algorithm. Constraint-based. Spirtes, Glymour & Scheines,
  *Causation, Prediction, and Search* (MIT Press, 2nd ed., 2000).
- `grasp`: Greedy Relaxations of the Sparsest Permutation. Lam, Andrews &
  Ramsey, UAI 2022 (arXiv:2206.05421).
- `boss`: Best Order Score Search. Andrews et al., NeurIPS 2023
  (arXiv:2310.17679).

Valid contributions may use constraint tests, score search, ordering or
permutation search, hybrid criteria, or a new modular CPDAG estimation
strategy, provided the output respects the expected causal-graph
representation.
